# Maths Dragons

A simple static maths game for kids, built with plain **HTML, CSS and JavaScript** so it can be hosted directly on **GitHub Pages**.

## How to play

- Answer maths questions to earn **coins**.
- Build a 5-answer streak to earn **crystals**.
- Spend either currency to hatch a **random creature**.
- Progress saves automatically in the browser.

## Run locally

If you have Python installed:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish on GitHub Pages

1. Push these files to your GitHub repository.
2. Open the repo on GitHub.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select your main branch and the `/ (root)` folder.
6. Save, then wait for GitHub to publish the site.

Because this project is fully static, there is no build step needed.
